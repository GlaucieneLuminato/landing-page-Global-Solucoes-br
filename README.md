## 📌 Sobre o projeto

A Global Soluções é uma empresa especializada em Medicina e Segurança do Trabalho, oferecendo soluções completas para empresas de diversos segmentos.

Este projeto consiste no desenvolvimento de uma Landing Page moderna, responsiva e otimizada, criada com foco em transmitir credibilidade, facilitar o contato com clientes e apresentar os principais serviços da empresa de forma clara e intuitiva.

O projeto foi desenvolvido utilizando React, Vite e Tailwind CSS, seguindo boas práticas de componentização, reutilização de código e organização de pastas.
